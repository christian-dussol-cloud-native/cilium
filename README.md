## About This Repository

This repository contains learning materials and educational examples for **Cilium**, the eBPF-based networking, security, and observability solution for Kubernetes.

**Educational focus.** Real code and examples

### [carousel/](./carousel)
Collection of visual presentations (carousels) illustrating Cilium concepts.

### [cilium-first-sample/](./cilium-first-sample)
Complete hands-on tutorial to get started with Cilium, including:
- Cluster setup with Minikube
- Cilium installation and verification
- Network policies (L3/L4/L7)
- Hubble observability
- Kyverno governance
- Payment API demo (PCI-DSS compliant)
